Module 01 FOUNDATION AIエージェントの

Claude Code 入門 — コードは書かない、動かすだけ

「こうしてほしい」と日本語で伝えるだけ

Claude Codeとは、AIに日本語で話しかけるだけで、プログラムを書いて動かしてくれるツールです。コードを読む必要も、書く必要も一切ありません。あなたがやることは、指示を出すことだけ。
